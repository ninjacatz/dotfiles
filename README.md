A collection of dotfiles and userscripts designed to work for a specific configuration of antiX-Core Linux running on a Thinkpad T420.

All userscripts start with the prefix 'qq' for easy identification and search.

Relying primarily on the following programs (but also others):
- Alacritty
- Vim
- Vifm
- MPV
- BSPWM
- BASH
- Bleachbit
- luckyBackup
